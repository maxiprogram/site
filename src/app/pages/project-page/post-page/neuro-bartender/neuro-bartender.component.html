NeuroBartender

Описание проекта

NeuroBartender – приложение работающее с устройством NeuroSky MindWave.

https://store.neurosky.com/pages/mindwave

Скорость налития рюмки зависит от показателей Attention получаемого от устройства -

распознающего мозговые волны, определяя эмоциональные состояния с помощью одного датчика на лбу.

Используемые технологии и языки программирования
  • Html/Css/JavaScript
  • Core WebView Chromium
  • NeuroSky MindWave SDK
  • C#
