AiArt создание изображения

Описание проекта

Кроссплатформенное, клиент-серверное приложение, для создания AI изображения из наброска.

    Список возвожных стилей, для генерации изображения:
  • 3D
  • Аналоговый фильм
  • Aниме
  • Кинематографическии
  • Комикс
  • Digital-арт
  • Фэнтези-арт
  • Изометрический
  • Штрих-арт
  • Низкополигональная
  • Неон-панк
  • Оригами
  • Фотографический
  • Пиксель-арт

Используемые технологии и языки программирования
  • Qt Framework
  • C/C++
  • Сторонний HTTP API-AI
