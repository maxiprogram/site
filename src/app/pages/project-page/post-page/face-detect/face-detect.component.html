FaceDetect фотобутка

Описание проекта

FaceDetect фотобутка – приложение работающее с видеопотоком камеры, распознает лицо, и эмоции найденого лица.

Функциональные возможности:

  • Распознание лица в реальном времени
  • RealTime preview c камеры(поддержка Web-камер и фотоаппаратов серии Canon)
  • Распознание эмоций по найденному лицу и вывод на экран(например гнев, презрение, отвращение, страх, счастье, нейтральное выражение, грусть и удивление)
  • Печать фото на принтер с предсказанием(предсказание генерируется исходя из базы данных по эмоциональному окрасу лица распознанного человека)

Используемые технологии и языки программирования
  • OpenCV
  • Azure Face API
  • C#
